About Me



Hello my name is James, I'm a Junior Front-End Web Developer based in South-East England. After completing developer bootcamp at Enspiral Dev Academy in Auckland NZ, I've gained a new appreciation for back-end development as well as my first love which is front-end UI design. I have experience with pair programming and working on dev teams utilising agile practices. I'm also familiar with the use of test-driven-development and object oriented programming to build responsive web apps.
