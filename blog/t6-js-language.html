JS Language

parentheses ( )
Parentheses in JavaScript are used for function calls, to surround conditional statements, or for grouping to enforce Order of Operations.
function myFunc()
if (condition1)
if ( (1 + 2) * 3)

brackets [ ]
Brackets are typically mostly used for accessing the properties of an Object (or the elements of an Array), so mylist[3] fetches the fourth element in the Array.
var mylist = [1,2,3,4];
alert(mylist[2]);

braces { }
Braces are used during the declaration of Object Literals, or to enclose blocks of code (function definitions, conditional blocks, loops, etc).
var objLit = { a: "I am an Object Literal." };

single quotes ' '
We can use single quotes to make a string. When using bracket notation you need to use single quotes to access object properties. To create a quote inside a string we use single quotes.
prompt('Hello!');
species['Human']
var pet = "She called her dog 'Bilbo Baggins'."

double quotes " "
Double quotes are most often used in JavaScript to denote strings, especially ones that contain apostrophes. And they are used to access object properties with bracket notation.
var describe = "He's tall."
var chocolate["Twix"]
