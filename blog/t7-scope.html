Scope

What is 'scope', and how does it work in JavaScript?
Scope is a way of assigning variables to the entirety of your JavaScript file (global scope) or to a specific function (local scope). Scope defines the part of the program where a particular variable is accessible. Imagine that our JavaScript file is the planet Earth. Cities, countries and even continents are like JavaScript functions with local variables, even with the continent sized functions. However these are all surrounded by the oceans which can’t be described as local because they wrap around everything else, therefore it’s called a global variable.

Here’s an example of a global & local variable (test): 

                var test = "I'm global";

                function testScope() {
                  var test = "I'm local";

                  console.log (test);
                }

                testScope();           // output: I'm local

                console.log(test);     // output: I'm global
